# Pantalla 09 — Meta cumplida
**ID:** meta-logro
**Canvas:** Mobile 375px
**Actor:** Adolescente (pantalla principal) + Padre (notificación push — referencia)
**Posición en el flujo:** Aparece automáticamente cuando la meta llega al 100%

---

## Prompt para Stitch

Diseña una pantalla de celebración mobile (375px) para una adolescente de 16 años que acaba de cumplir su primera meta de ahorro en una app financiera. La celebración debe sentirse real y motivadora — sin ser exagerada ni infantil.

**Layout:** Pantalla completa. Parte superior grande para la celebración visual. Parte inferior para las acciones.

**Parte superior (aprox. 55% de la pantalla):**
- Fondo con gradiente verde muy suave (blanco a verde clarísimo)
- Insignia grande centrada: círculo verde con ícono de trofeo o estrella, tamaño 80px
- Badge bajo la insignia: "Meta cumplida" en verde
- Título grande: "¡Lo lograste!"
- Subtítulo: "Llegaste a tu meta 'Mis AirPods'. ¿Qué quieres hacer con esta plata?"

**Parte inferior (acciones):**
- Tres opciones de acción en columna:
  1. Botón primario verde: "Retirar"
  2. Botón secundario outlined: "Crear una nueva meta"
  3. Link texto verde (no botón): "Hacer crecer esta plata →"
- Texto pequeño gris bajo el botón "Retirar": "Para retirar, tu equipo recibirá una notificación para aprobarlo."

**Estilo visual:**
- Momento de logro: debe sentirse especial pero sobrio — no cartoon
- El verde es dominante en la parte superior (celebración), blanco en la inferior (acción)
- La insignia/trofeo es el elemento emocional más importante
- Energético pero no infantil — un adolescente de 17 años debe querer compartir esta pantalla

## Restricciones de marca Skandia

- Insignia: círculo `#00C73D`, ícono blanco, sombra verde `0 4px 20px rgba(0,199,61,0.3)`
- Badge "Meta cumplida": fondo `#E8F8ED`, texto `#00703A`, `border-radius: 20px`
- Título "¡Lo lograste!": Montserrat Bold 32px, color `#00703A`
- Botón primario: verde `#00C73D`, ancho 100%, altura 52px
- Botón secundario: borde `1px solid #00C73D`, texto `#00C73D`, fondo blanco, ancho 100%
- Link: texto `#00C73D`, sin subrayado, con flecha →
